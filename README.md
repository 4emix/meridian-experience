# ÇİMTAŞ — Turn Your Dreams Into Real

An award-style immersive web experience for **Çimtaş İnşaat** (Ankara, est. 1997) — architecture, engineering & construction.

## The experience
- Full-screen architectural **blueprint** intro with the headline _TURN YOUR DREAMS INTO_ and an interactive **REAL** button.
- Clicking **REAL** plays a cinematic morph (Higgsfield-generated) that assembles the 2D plan into a furnished 3D isometric apartment.
- Seamless reveal into a premium scrolling site: hero, expertise, pinned horizontal projects, process, and contact.

## Stack
Next.js 14 · React 18 · TypeScript · Tailwind CSS · GSAP + ScrollTrigger · Lenis · Framer Motion.

## Develop
```bash
npm install
npm run dev   # http://localhost:3007
```

## Deploy
Zero-config on **Vercel** — import the repo and deploy. Build: `next build`.

Imagery and the intro morph video were generated with Higgsfield.
